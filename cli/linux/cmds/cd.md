切换目录  

## 使用

1. 切换到指定路径 : `cd path_directory/`
2. 切换到上一层目录 : `cd ..`
3. 回到用户主目录 : `cd ~`
4. 回到上一个工作目录 :  

		$ pwd 
		/home/vagrant/synced_folder/WorkSpace/micro_knows
		$ cd ~
		$ pwd
		/home/vagrant
		$ cd -
		/home/vagrant/synced_folder/WorkSpace/micro_knows
