基本语法

2015-09-28 03:07:23 by MwumLi

  1. 每个语句以 ; 结束
    JS 并不强制在语句末尾添加 ;, 浏览器中 js 解释器会自动在每个语句末尾补上 ;

  2. 语句块是一组语句的集合, 用 {...} 包含

  3. 可以使用缩进来表现代码的层次

  4. js 本身对嵌套的层级没有限制, 但是建议不要使用过多的嵌套, 增加代码阅读的难度

  5. js 严格区分大小写, 如果弄错大小写, 程序将出错或者运行不正常

注释

  1. 以 // 开头直到行末的字符被视为行注释

  2. 用 /*...*/ 把多行字符包裹起来, 这是块注释
