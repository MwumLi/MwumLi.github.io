# file-timestamps

Linux 下会为文件的记录三种时间 : atime 、mtime 、ctime.
可以通过 ls 去查看  
可以通过 touch 去修改 

# atime

time of last access

# mtime

time of last modification

# ctime

time of last status change

# 通配符
